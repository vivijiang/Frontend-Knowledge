Responsive Design
---------

**Media Queries**


> **Action in Bootstrap**
> - @media (min-width: 992px) and (max-width: 1199px) { ... }
> - @media print { ... }
> - @media screen and (orientation: landscape)
> - @media screen and (orientation: portrait) 

**Responsive Images**
> **Action in Bootstrap**
> - img {max-with: 100%;}
> - img data-fullsrc=""

**Meta**
